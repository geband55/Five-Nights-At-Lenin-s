# Инструкция по получению билда игры

Данный репозиторий устроен следующим образом:
- **Код игры** хранится в обычном Git.
- **Собранный билд** хранится в Git LFS (Large File Storage).

Чтобы получить актуальную версию билда, следуйте инструкциям ниже.

## 1. Установите Git (если ещё не установлен)

Если у вас нет Git, скачайте и установите его с официального сайта:  
[https://git-scm.com/downloads](https://git-scm.com/downloads)  

После установки проверьте, что Git работает, выполнив в терминале:

```shell
git --version
```

## 2. Установите Git LFS (если ещё не установлен)

Git LFS необходим для загрузки файлов билда. Установите его по инструкции с официального сайта:
[https://git-lfs.com]

После установки выполните команду для инициализации Git LFS в вашей системе:

```shell
git lfs install
```

## 3. Склонируйте репозиторий (первый раз)

Клонируйте репозиторий по HTTPS (нажмите на гитхабе зелёную кнопочку ```code``` и скопируйте <ссылку> из раздела HTTPS). Команда:

```Shell
git clone <ссылка>
```

После клонирования перейдите в папку проекта (она появится после клонирования):

```Shell
cd <папка с репозитоием>
```

Теперь выкачайте файлы билда, которые хранятся в Git LFS:

```Shell
git lfs pull
```

Примечание: Без выполнения ```git lfs pull``` вместо реальных файлов билда вы увидите только текстовые указатели (pointer files). После выполнения команды будут загружены все LFS-объекты, включая билд.

## 4. Последующие обновления

Когда вам нужно получить последние изменения (как код, так и билд), выполните:

```Shell
git pull
git lfs pull
```

Первая команда обновит локальный код и метаданные LFS, вторая — загрузит обновлённые файлы билда.
